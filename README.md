## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript
with GraphQL, Fastify, Prisma, MySQL, Docker
starter Template repository.

## Getting Started

```bash
$ docker-compose up
```

to Start Server.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
